// how a project actually moves

Five stages, start to deploy

This is the real order, not a sales pitch — each stage produces something the next one needs.

  1. 01

    Discover

    A call, a form, or both. We learn what the site needs to do before anyone opens an editor.

  2. 02

    Design

    Wireframes and a style guide, reviewed with you before a single line of code is written.

  3. 03

    Develop

    Semantic HTML, styled with care, wired up with just enough JavaScript to do the job.

  4. 04

    Deploy

    Pushed to production and tested on real devices — not just a laptop at 1920×1080.

  5. 05

    Support

    A maintenance window after launch, because real sites need upkeep, not just a launch date.
